These case files are for drive-cycle simulation of the LFP batteries under the US06 protocol at different temperatures, presented in Fig. 3b and 3d. The US06 protocol is implemented in the model as user defined load profiles (UDLP).

Case-file Name: Blade_LFP_US06_40kWh_T60.xml
Descripton: ambient temperature at 60oC

Case-file Name: Blade_LFP_US06_40kWh_T40.xml
Descripton: ambient temperature at 40oC

Case-file Name: Blade_LFP_US06_40kWh_T25.xml
Descripton: ambient temperature at 25oC

Case-file Name: Blade_LFP_US06_40kWh_T10.xml
Descripton: ambient temperature at 10oC

Case-file Name: Blade_LFP_US06_40kWh_T00.xml
Descripton: ambient temperature at 0oC

Case-file Name: Blade_LFP_US06_40kWh_TM10.xml
Descripton: ambient temperature at -10oC

Case-file Name: Blade_LFP_US06_40kWh_TM20.xml
Descripton: ambient temperature at -20oC

File name: LFP_US06_40kWh_wRegen
Descripton: Load profile of battery power for the US06 protocol with consideration of regenerative braking

File name: LFP_US06_40kWh_NoRegen
Descripton: Load profile of battery power for the US06 protocol without consideration of regenerative braking
